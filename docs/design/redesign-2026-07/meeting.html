Resumo

A equipe fechou a arquitetura de providers do resumo: o built-in local continua como padrão, com map-reduce por blocos de 20 minutos para reuniões acima de 40 minutos. Provedores externos (Claude, OpenAI, Groq, OpenRouter) e o modo CLI Agent permanecem como escolha explícita do usuário — sem fallback automático.

Decisões

  • Map-reduce vira o caminho padrão para transcrições longas; consolidação final no mesmo provider.
  • Sem fallback automático entre providers: falhou, mostra o erro e o usuário decide.
  • Nomes de falantes só entram no prompt quando o provider roda localmente.
